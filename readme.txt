
			Etapes pour integrer l'equipe de Travail OPW.
Prerequis
	- Installer Github pour windows https://desktop.github.com
	- Installer Netbeans
	- Installer Wampserver
	-


Faire une copie du repertoire de travail pour participer au versionnage du code ( Developpement en temps réel )
	
Comment s'y prendre?
	- Ouvrir l'application Github et se connecter 
	- Ouvrir son navigateur et aller sur github.com, puis se connecter 
	- Dans la barre de recherche, Tapez le mot clé suivant " opw "
	- Dans la liste de resultat, Selectionner " guyaristide/opw "
	- Une fois la page chargée, cliquer sur l'icone " Fork "
	- Revenir dans le logiciel Github, cliquez sur l'icone "+" en haut a gauche
	- Ensuite cliquez sur " clone "
	- Selectionner le projet "opw" puis clique sur "clone repository"
